# Elite Paint Voucher Entry System - README

## 🎫 সিস্টেম পরিচয়

এলিট পেইন্ট ভাউচার এন্ট্রি সিস্টেম একটি আধুনিক, পূর্ণ-বৈশিষ্ট্যসম্পন্ন ক্লাউড-ভিত্তিক সমাধান যা:

- ✅ বহু-ব্যবহারকারী সমর্থন সহ
- ✅ সম্পূর্ণ বাংলা ভাষা সমর্থন
- ✅ নিখুঁত প্রিন্ট লেআউট (অফসেট সহ)
- ✅ স্বয়ংক্রিয় সুপারিশ বৈশিষ্ট্য
- ✅ QR কোড জেনারেশন এবং শেয়ারিং
- ✅ প্রশাসক ড্যাশবোর্ড এবং ব্যবহারকারী ব্যবস্থাপনা

## 🏗️ আর্কিটেকচার

```
Frontend (Cloudflare Pages) → Backend (Cloudflare Workers) 
                              ↓
                    Database (D1) + Cache (KV)
```

## 🚀 দ্রুত শুরু

### ইনস্টলেশন

```bash
git clone https://github.com/ronyopq/elite-paint-voucher-entry-system
cd elite-paint-voucher

# Setup script চালান
./scripts/setup.sh          # Linux/Mac
./scripts/setup.bat         # Windows
```

### পরিবেশ কনফিগারেশন

`.env.local` তৈরি করুন:

```env
GOOGLE_CLIENT_ID=your_client_id
GOOGLE_CLIENT_SECRET=your_client_secret
GOOGLE_REDIRECT_URI=https://yourdomain.com/auth/callback
APP_DOMAIN=https://yourdomain.com
SUPER_ADMIN_EMAILS=admin@example.com
SESSION_SECRET=random_secret
```

### ডেটাবেস সেটআপ

```bash
# বিকাশ মোডে
wrangler d1 create elite-voucher

# মাইগ্রেশন চালান
wrangler d1 execute elite-voucher --local < migrations/001_initial_schema.sql
```

### উন্নয়ন শুরু করুন

```bash
npm run dev
# http://localhost:8787 এ আপনার ব্রাউজারে যান
```

## 📊 মূল বৈশিষ্ট্য

### 🔐 প্রমাণীকরণ
- Google OAuth 2.0 ইন্টিগ্রেশন
- সুরক্ষিত সেশন ব্যবস্থাপনা
- ভূমিকা-ভিত্তিক অ্যাক্সেস নিয়ন্ত্রণ

### 🎫 ভাউচার ম্যানেজমেন্ট
- দ্রুত ভাউচার তৈরি এবং সংরক্ষণ
- স্বয়ংক্রিয় ভাউচার নম্বর বৃদ্ধি
- আংশিক সংরক্ষণ এবং পুনরায় সম্পাদনা
- সীমাহীন ভাউচার প্রিন্ট করুন

### 💾 স্মার্ট পরামর্শ
- স্বয়ংক্রিয় শ্রেণীকরণ পরামর্শ
- ব্যবহার-ভিত্তিক র্যাঙ্কিং
- দ্রুত ডেটা এন্ট্রি

### 🖨️ নিখুঁত প্রিন্টিং
- সঠিক 797×542px ক্যানভাস আকার
- হাতে তৈরি ফিল্ড স্থান নিয়ন্ত্রণ
- বহু-পৃষ্ঠা সমর্থন
- অফসেট সামঞ্জস্য

### 🔗 ডিজিটাল শেয়ারিং
- প্রতিটি ভাউচারের জন্য অনন্য পাবলিক লিঙ্ক
- QR কোড সহ শেয়ারযোগ্য লিঙ্ক
- পড়া-একমাত্র পাবলিক ভিউ
- দৃশ্য ট্র্যাকিং

### 📊 রিপোর্ট
- মাসিক এবং সাপ্তাহিক সারসংক্ষেপ
- ব্যবহারকারী কার্যকলাপ রিপোর্ট
- রাজস্ব বিশ্লেষণ
- শীর্ষ ব্যবহারকারী পরিসংখ্যান

### 👨‍💼 অ্যাডমিন ড্যাশবোর্ড
- সমস্ত ব্যবহারকারী এবং ভাউচার দৃশ্য
- ব্যবহারকারী ব্লক/আনব্লক
- ট্রায়াল সম্প্রসারণ
- বৈশ্বিক সেটিংস পরিচালনা
- অডিট লগ দৃশ্য

## 🔄 কর্মপ্রবাহ

### ভাউচার তৈরি
1. লগইন (Google OAuth)
2. নতুন ভাউচার ট্যাবে যান
3. তথ্য পূরণ করুন
4. স্বয়ংক্রিয় পরামর্শ ব্যবহার করুন
5. সংরক্ষণ এবং প্রিন্ট করুন

### ভাউচার শেয়ার করুন
1. ভাউচার প্রিভিউতে "শেয়ার" ক্লিক করুন
2. অনন্য লিঙ্ক অনুলিপি করা হয়
3. যেকোনো জায়গায় ভাগ করুন
4. প্রাপক পড়ার জন্য খোলে

### প্রশাসক ব্যবহার
1. Admin পয়েন্টে যান
2. ড্যাশবোর্ড দেখুন
3. ব্যবহারকারী পরিচালনা করুন
4. ট্রায়াল এবং সেটিংস পরিচালনা করুন
